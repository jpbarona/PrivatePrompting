This project is a proof of concept for privacy-aware split inference for large language models, based on Privacy-Aware Split Inference with Speculative Decoding for Large Language Models over Wide-Area Networks by Michael Cunningham (February 2026). The paper proposes splitting an LLM between a trusted local device and an untrusted remote machine so that raw tokens never leave the trusted side: token embedding and final token decoding stay local, while the compute-heavy middle layers run remotely and exchange only intermediate hidden states over the network.

The problem we are trying to solve is the tradeoff between capability and privacy. Stronger models often require remote compute, but sending raw prompts and outputs to external infrastructure creates a trust and data exposure problem. This project explores whether inference can be partitioned so that sensitive text remains local while still benefiting from outside compute.

Our current solution is to build the architecture in phases. So far, we have shown locally that split inference can work end to end while preserving exact greedy decoding behavior: the trusted side keeps tokenization, embeddings, and output decoding local, while a separate middle stage processes hidden states only. This is an early proof of concept for the paper’s broader idea, not yet a full deployment.

There is still substantial work ahead. To make this a real system, we need to move from a local prototype to independent nodes communicating over an encrypted channel, deploy it across a private network, implement a stronger local-prefix / remote-middle / local-suffix split that matches the paper more closely, and build a frontend that makes the system usable in practice. The long-term goal is a practical inference system that preserves much of the usefulness of remote compute while giving users stronger control over where sensitive data is exposed.
